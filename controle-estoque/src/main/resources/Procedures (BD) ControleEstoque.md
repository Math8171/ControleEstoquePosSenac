**1. Tabela Estoque:**

SQL
```
CREATE TABLE Estoque (
  ID_ESTQ NUMBER PRIMARY KEY,
  FK_ID_PROD NUMBER NOT NULL REFERENCES Produto(ID_PROD),
  ESTQ_QTDE_TOTAL_PROD NUMBER NOT NULL,
  ESTQ_QTDE_MIN_ESTOQUE NUMBER NOT NULL,
  ESTQ_VALOR_MEDIO NUMBER NOT NULL,
  ESTQ_VALOR_TOTAL NUMBER NOT NULL
);
```

PL/SQL
```
CREATE OR REPLACE PROCEDURE create_estoque_table IS
BEGIN
  EXECUTE IMMEDIATE '
    CREATE TABLE Estoque (
      ID_ESTQ NUMBER PRIMARY KEY,
      FK_ID_PROD NUMBER NOT NULL REFERENCES Produto(ID_PROD),
      ESTQ_QTDE_TOTAL_PROD NUMBER NOT NULL,
      ESTQ_QTDE_MIN_ESTOQUE NUMBER NOT NULL,
      ESTQ_VALOR_MEDIO NUMBER NOT NULL,
      ESTQ_VALOR_TOTAL NUMBER NOT NULL
    )
  ';
END;
/

**5. Tabela Movimentação:**

SQL
```

  ID_MOV NUMBER PRIMARY KEY,
  FK_ID_PROD NUMBER NOT NULL REFERENCES Produto(ID_PROD),
  FK_ID_LOJA NUMBER NOT NULL,
  FK_ID_TP_MOV NUMBER NOT NULL REFERENCES TP_Movimentacao(ID_TP_MOV),
